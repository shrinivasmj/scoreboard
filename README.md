# 🏀 Score Display

A simple and responsive score display application built using **HTML, CSS, and JavaScript**. This project allows users to keep track of scores with a clean and easy-to-use interface.

## 🚀 Features

- Display scores for two teams/players
- Interactive score update buttons
- Clean and responsive user interface
- Beginner-friendly JavaScript project
- Lightweight and fast

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
score-display/
│── index.html
│── score-display.css
│── score-display.js
│── ground.png
│── Digital-Numbers.ttf.woff
```

## 🎯 Purpose

This project was created to practice JavaScript DOM manipulation, event handling, and basic front-end development concepts.
